# NativeFileDialog Rust bindings

Experimental bindings to [nativefiledialog](https://github.com/mlabbe/nativefiledialog).

## Supported platforms
* [x] OS X (10.11)
* [x] Linux x86_64 (Ubuntu 15.04)
* [ ] Linux x86_32
